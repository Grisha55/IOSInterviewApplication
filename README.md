# IOSInterviewApplication

## This ios app is for help people prepare for an interview

## What technologies have been applied in this app? 
- Interface design with code
- GCD
- TDD tests
- MVP and MVVM architectures
- RxSwift
- RxCocoa
- Realm
- Firebase (registration) 
- Custom UI elements
- Custom animations
